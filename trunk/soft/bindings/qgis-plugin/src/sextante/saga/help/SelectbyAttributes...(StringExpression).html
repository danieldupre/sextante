Select by Attributes... (String Expression)

V.Olaya (c) 2004, O.Conrad (c) 2011
Searches for an character string expression in the attributes table and selects records where the expression is found.

Parameters

Shapes
Input Shapes
Attribute
Table field
attribute to be searched; if not set all attributes will be searched
Expression
Text
Case Sensitive
Boolean
Select if...
Choice
Available choices: attribute is identical with search expression, attribute contains search expression, attribute is contained in search expression
Method
Choice
Available choices: new selection, add to current selection, select from current selection, remove from current selection
